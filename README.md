## Instructions for running locally
Requires Node.js, npm

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Generate Documentation
```
npm run docs
```
